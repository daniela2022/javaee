# javaee
Java Web - Agenda de contatos - Cabeleireira Leila

Projeto idealizado para desenvolver um CRUD -Create,Read,Update and Delete em Java.

Utilizando as seguintes tecnologias:

- Servidor TomCat
- JavaEE
- Banco de dados MySQL
